# 远征战斗第一阶段调优说明

## 目标

本阶段优先修复会直接破坏“搜打撤”决策的规则漏洞和交互问题，覆盖：地图可达性、镜头跟随、武器切换公平性、伤害提示、搜索产量、主动撤退、返程压力、双撤离价值及整备/局内界面分离。

本阶段不扩充新地图与新物品，先保证现有循环的规则可信、风险可读、操作可验证。

## 已完成修改

### 1. 地图导航与镜头

- 新局会检测出生区东向出口，自动移开可能封住首段道路的西侧废墟。
- 增加基于 40px 角色占地的网格可达性检查；极端障碍布局会生成保底东向通道。
- 旧存档会尽量保留原有地形，只有确认无法向东通行时才执行安全修正。
- 镜头平滑追踪增加安全框约束，掉帧、后台恢复或连续输入时角色不会离开可见视口。
- 未侦察路线统一显示“未知信号”，主 HUD、路线卡和追踪结果不再提前泄露地点名称。
- 路线切换统一返回追踪目标和目标 ID，UI 会立即显示选中状态和实时距离。

### 2. 武器与战斗公平性

- 三把武器改为独立射击冷却，切枪不会缩短原武器剩余冷却。
- 切枪增加 350ms 举枪锁；精确枪 920ms 射击间隔无法通过来回切枪绕过。
- 切走正在换弹的武器会取消换弹；非当前武器不能后台换弹。
- 局内切换只改变当前战斗模式，不再回写局外配装或活动远征快照。
- Boss 范围攻击固定显示 150px 警示外圈，实际伤害半径为 145px；脉冲仅影响透明度和内圈。
- 普通、精英和 Boss 使用与可见贴图一致的显式命中盒，覆盖约 75% 可见躯干。
- 新版活动局保存各武器冷却和换枪锁；旧版共享冷却存档会自动迁移。

### 3. 搜索、返程和撤离

| 规则 | 修改前 | 修改后 |
| --- | --- | --- |
| 仔细搜刮 | 12秒、威胁+14、伏击30%、单容器2–3件 | 8秒、威胁+10、伏击20%、单容器1–2件 |
| 宠物侦察 | 单容器2件 | 单容器1件，保留稳定品质优势 |
| 普通搜索点总产量 | 可能超过背包节奏 | 2–4件 |
| 密封仓库总产量 | 可能一处接近或超过8格背包 | 3–5件 |
| 应急撤离 | 11秒、基础6敌人、固定额外精英 | 9秒、基础5敌人、不固定追加精英 |

- 旧版无实体容器的同步搜索入口继续按整个地点结算，保持兼容。
- 返程压力不再依赖 `canExtract()`，待处理战利品无法绕过压力。
- 只有角色实际缩短到已解锁撤离点的距离时才累计返程压力；继续向深区移动不会误计。
- 撤离路线卡新增预计守点时间和敌人数，并保留距离与补给成本。
- 密封安全袋世界事件每局最多一个，启用时威胁 +5，不再是无成本纯收益。

### 4. 失败、整备与局内信息

- 主动撤退保留 30% 战斗金币和 40% 经验。
- 阵亡只保留 10% 战斗金币和 20% 经验，主动止损不再劣于故意送死。
- “保险格”在玩家界面统一改名为“安全袋”，与未来的局外装备保险区分。
- 战斗、搜索和撤离守点期间禁止瞬时整理安全袋。
- 主动放弃按钮第一次点击会切换为醒目的“再次点击确认放弃”，3.2 秒后自动取消确认。
- 结算后先进入独立整备状态，再由玩家确认配装并开始下一局。
- 局内隐藏仓库和委托板，只保留本局携带、战斗模式、风险、路线和背包。
- 委托板显示具体奖励，并允许在整备阶段放弃已接委托。
- 风险预览显示主动撤退/阵亡现金差异、安全袋价值和可能丢失的护甲或宠物链接器。
- 非战斗阶段禁用射击与换弹按钮；武器卡增加“稳定中距 / 近距爆发 / 远距精确”定位提示。

## 验证结果

- `npm test`：全部源码语法检查和 Node 测试通过。
- 新增或扩充回归覆盖：
  - 镜头安全框与世界边缘约束。
  - 2000 个地图种子的出生区可达性。
  - 未知地点信息遮蔽与追踪状态。
  - 独立武器冷却、换枪锁、换弹取消和旧存档迁移。
  - Boss 警示/伤害边界与敌人命中盒。
  - 待选战利品不能绕过返程压力。
  - 搜索地点总产量、撤退/阵亡收益和应急撤离参数。
- 应用内浏览器复玩：整备、开始远征、路线切换、长距离移动、局内切枪、主动撤退和返回整备均通过。
- 浏览器控制台未发现 warning 或 error。

## 后续阶段建议

1. 开局一次生成 8–12 个持久 POI 和连接关系，替代当前逐层二选一的线性分叉。
2. 将搜索暴露值接入真实巡逻和 35%/70% 进度预警，允许暂停或提前撤离搜索。
3. 重平衡三把武器的 DPS、弱点和穿透，并提高敌人并发、追击与包抄能力。
4. 重标浅层稀有/史诗掉率，补充装备保险、材料消耗和撤离后整理流程。
5. 将静态委托拆为一次性主线链与分层可重复支线，并补齐刷新与目标宠物选择。
