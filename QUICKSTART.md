## ⚡ 5分钟快速上手

### 第一步：下载项目

```bash

git clone https://github.com/Chacat68/PetPlan.git
cd PetPlan

# 或直接下载 ZIP 文件并解压
```

### 第二步：启动游戏

#### 方法一：Python 3（推荐）
```bash
python -m http.server 8080
```

#### 方法二：Node.js
```bash
npx http-server
```

#### 方法三：VS Code Live Server
1. 安装 Live Server 扩展
2. 右键点击 `index.html`，选择 "Open with Live Server"

### 第三步：打开浏览器

访问 `http://localhost:8080`

**完成！** 🎮 游戏应该已经加载。

## 🎮 游戏玩法指南

### 1️⃣ 角色升级

**升级属性**
- 💪 **攻击力**: 增加伤害输出
- ❤️ **生命值**: 增加生存能力
- 🛡️ **防御力**: 减少伤害受到
- 💚 **生命恢复**: 自动回血
- 💥 **暴击伤害**: 暴击时的额外伤害
- ⚡ **攻击速度**: 更快的射击速度
- 🎯 **暴击率**: 触发暴击的概率
- 🌀 **多重射击**: 一次射多发

**升级方法**
- 点击属性旁的 `+` 按钮升级（支持长按快速升级）
- 打怪物获得金币进行升级
- 使用快捷菜单批量升级 (+1, +10, +100)

### 2️⃣ 战斗系统

**自动战斗**
- 游戏会自动生成怪物
- 你的子弹会自动射击
- 消灭怪物获得金币和经验

**战斗属性**
- 伤害计算: 攻击力 × 暴击倍率（如果暴击）
- 防御减伤: 伤害 × (1 - 防御力%)
- 自动回血: 每秒恢复生命值

### 3️⃣ 宠物系统

**宠物收集**
1. 点击底部 **宠物** 标签页
2. 查看已收集和未收集的宠物
3. 点击宠物可查看详细信息

**宠物编队**
- 最多可编队 3 只宠物
- 编队的宠物可以参与战斗
- 不同稀有度的宠物属性不同

**宠物养成**
- 消耗金币升级宠物等级
- 升级后增加属性加成
- 稀有度更高的宠物加成倍数更大

### 4️⃣ 领地系统

**访问领地**
1. 点击底部 **领地** 标签页
2. 或在主菜单点击 **领地** 按钮

**建筑管理**
- 点击空地块建造建筑
- 点击已建建筑升级
- 建筑会自动产出资源

**资源产出**
- 🏭 工厂: 产出金币
- 🔮 魔法塔: 产出水晶
- 📚 图书馆: 增加经验获取

**领地扩张**
- 通过升级主基地解锁新地块
- 使用金币和水晶扩张领地
- 解锁更多建筑位置

### 5️⃣ 存档系统

**保存游戏**
1. 点击 **💾 存档/读档** 按钮
2. 选择要保存的槽位（1-5）
3. 点击 **保存** 按钮

**加载游戏**
1. 打开存档菜单
2. 选择要加载的存档
3. 点击 **加载** 按钮

**导入导出**
- **导出**: 下载存档为 JSON 文件备份
- **导入**: 从 JSON 文件恢复存档

### 6️⃣ 成就系统

**每日任务**
1. 点击底部 **📅 收益** 按钮
2. 查看当日任务进度
3. 完成任务后点击 **领取** 获得奖励

**长期成就**
- 击杀怪物、积累金币、升级属性等
- 完成成就获得额外奖励
- 每个成就只能领取一次

## 🐛 常见问题

### Q1: 游戏无法加载？
**解决方案**:
1. 刷新页面 (Ctrl+R 或 Cmd+R)
2. 清空浏览器缓存
3. 使用 Chrome 或 Firefox 最新版本
4. 访问 [诊断页面](diagnostic.html) 检测问题

### Q2: 存档无法保存？
**解决方案**:
1. 检查浏览器是否允许 LocalStorage
2. 退出隐私浏览模式
3. 检查浏览器存储空间是否满
4. 尝试导出存档到文件备份

### Q3: 游戏运行缓慢？
**解决方案**:
1. 关闭其他应用
2. 清空浏览器缓存
3. 禁用浏览器扩展
4. 降低屏幕分辨率

### Q4: 宠物无法编队？
**解决方案**:
1. 确认宠物已解锁（等级足够）
2. 最多只能编队 3 只宠物
3. 点击宠物卡片选择编队
4. 刷新页面重试

### Q5: 如何导入已有存档？
**解决方案**:
1. 打开存档菜单
2. 切换到 **导入存档** 标签页
3. 选择之前导出的 JSON 文件
4. 选择目标槽位
5. 点击导入按钮
