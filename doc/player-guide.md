# PetPlan 玩家指南

## 开始游戏

PetPlan 使用横屏舞台。请通过本地服务器打开，并在移动设备上旋转为横屏。

默认进入命运场景：中央是桌面硬币，右侧是按当前状态整理的商店，左侧 HUD 用于切换命运与领地，并打开宠物和成就。远征不再提供全局直达按钮，需要先进入领地，再前往西侧远征入口。

## 命运桌

点击桌面硬币会结算正面或反面。两类资源用于购买不同成长项：

- **购买硬币**：增加桌面可点击目标。
- **结算芯片**：提高自动结算收益。
- **小助手**：自动前往硬币并结算。
- **银色点击**：提高人工点击收益。
- **金色引擎**：缩短自动结算间隔。
- **主角训练 / 宠物训练**：提升后续战斗表现。

优先查看右侧“推荐”和中央“下一目标”。推荐会随资源、自动化与领地进度变化；下一目标按钮可以直接前往对应场景，被资源阻塞时会写明还差多少金币和水晶。

首次进入时，命运桌还会依次提示翻转、扩桌、招募助手和建造第一座建筑。它只说明下一步，不会限制你购买其他项目。

已投入的升级会形成“先攻”“协同”或“拓域”倾向。倾向会帮助推荐理解你的打法，但随时可以改投其他方向。

## 宠物远征

战斗已改为“宠物远征·可操控大地图搜打撤”。每局都从地图西侧入口出发，拥有满生命、2 份补给和一个空背包。你需要亲自移动探索迷雾、前往地点、搜索或战斗，最后再返回入口撤离。

进入远征后会切换为沉浸战斗界面并隐藏基地 HUD。需要提前结束时按 **Esc** 打开退出确认：整备阶段会直接返回领地；安全行进阶段确认退出按“主动止损”结算，战斗或撤离守点中退出则按战败结算，不能把退出菜单当成无风险撤离。

### 移动、追踪与交互

- 使用 **WASD**、**方向键**或画面左下角的**屏幕方向键**移动。斜向输入会自动归一化，不会获得额外速度。
- 点击顶部**目标条**或按 **M**，会在当前路线、已发现事件和已解锁撤离点之间轮换追踪；角色不会自动寻路，也不会立即进入事件。战场画布上的鼠标或触摸输入用于瞄准射击。
- 走到地点附近后，按 **E** 或点击画面右下角的**交互**按钮，才会进入该地点、开始搜索或启动信标。
- 行动阶段不再保留右侧终端：搜索和营地的二选一操作只在当前情境出现，战斗与撤离守点只保留移动、射击、队伍技和补给。
- 岩石、水域和遗迹会阻挡移动，需要绕路前进。切换到其他主场景时，远征会暂停并保留当前页面会话中的局内状态。

### 大地图与迷雾

大地图会出现补给站、密封仓库、巡逻区、精英巢穴、安全屋和核心守卫。最初只有入口附近清晰可见；随着角色行进，主画面和小地图上的迷雾会逐步揭开，附近地点也会显示提示。

小地图会显示已探索范围、角色位置、已发现地点、当前追踪目标和入口撤离信标。尚未揭开的区域不会提前显示完整地形，因此追踪只是导航辅助，实际路线仍需自行探索。

不同地点会触发不同事件：

- **搜索区域**：选择搜索方式取得战利品，也可能遭到伏击。
- **战斗区域**：清理全部敌人后取得奖励并继续规划路线。
- **安全屋**：可以恢复生命、降低威胁并补充 1 份补给，也可以直接离开。
- **精英与核心区域**：敌人更强，但更容易获得高品质战利品。

每个搜索地点只需要选择一次：

- **快速拿取**：耗时短、获得 1 件物资，警戒和伏击风险最低。
- **彻底搜刮**：耗时较长、获得更多物资，但会明显提高警戒和伏击概率。

宠物侦察不再是一种额外按钮，而是自动生效的探索天赋。搜索按钮会在选择前显示物品数、警戒、补给率或伏击率的实际变化，结果提示会说明生效的宠物天赋；天赋有统一上限，因此三只同类宠物不会无限叠加收益。

### 战斗与宠物技能

出发前可以选择步枪、霰弹枪或精确枪，进入远征后该武器会锁定到本局。进入战斗后使用 **WASD** 走位，以鼠标或触摸战场瞄准并按住射击；空弹匣会自动换弹。移动只会扩大弹道散布，不会同时降低伤害和射速。

战斗时只显示队长宠物的一个主动队伍技能，冷却结束后可按 **Q** 或点击按钮释放。其余上阵宠物不占主动操作位，继续提供护盾、光环、救援和探索天赋。

生命会在本局中持续保留，不会在空地图行走时无限自然恢复。受伤后可以消耗补给恢复生命，也可以寻找安全屋休整。局内只需要关注 0～100 的一条警戒值；警戒越高，敌人组合、精英和撤离追兵越危险，同时奖励也会提高。

### 背包与撤离

远征背包最多暂存 8 件战利品。安全行进、营地或撤离准备阶段，可点击顶部“背包”或按 **B** 打开轻量抽屉，并在其中整理安全袋。背包已满时，系统会自动比较单一价值：更高价值物资替换最低价值且未受保护的物资，较低价值物资自动放弃；旧存档若仍有待处理物品，会强制打开取舍卡以避免流程卡死。撤离前，背包中的收益都不会进入局外库存。

清理至少 3 个地点后，西侧入口的撤离信标会解锁。此时可以继续深入争取稀有物品，也可以追踪入口、亲自走回信标附近，再按 **E** 或点击“交互”启动撤离。

入口撤离路程较远，但启动时间短、敌军压力较低且不消耗补给；继续深入后可以使用更近的应急撤离点，但需要消耗补给并承受更强追兵。撤离开始后需要留在信标范围内守住倒计时；离开撤离圈时，**只有撤离倒计时会暂停**，不会额外倒退，敌人和战斗仍会继续。

- **撤离成功**：带回全部背包战利品，并结算本局金币、水晶和经验。
- **安全阶段主动止损**：失去未保护战利品，仅保留较少战斗收益和安全袋物品。
- **战败或战斗中退出**：风险最高，只保留战败保底和安全袋物品。

需要提前结束时统一按 **Esc** 打开退出确认，不再在战斗界面放置第二个“放弃本局”入口。活动远征会保存路线、生命、武器弹药、背包和随机序列；刷新后可以继续当前远征。为防止复制战利品，活动远征期间不能用快速读档回退。每次结算后会自动保存。

局外资源仍用于长期成长：金币用于属性升级、宠物和建筑，红宝石用于稀有宠物，水晶用于领地建筑与扩张。

## 宠物

HUD 的“宠物”按钮打开管理页面。

- **编队**：最多上阵 3 只宠物。
- **背包**：查看并上阵已解锁宠物。
- **图鉴**：查看解锁要求与资源成本。

每只宠物卡会同时显示战斗技能、探索天赋和基地岗位。成功撤离会按探索深度提高上阵宠物的羁绊，失败也会获得少量羁绊；羁绊上限为 100，满值结算会显示“羁绊已满”。

宠物训练由命运商店触发，只提高宠物等级，不增加羁绊；羁绊只来自远征结算。上阵状态会影响队长技能、搜索天赋、战斗被动和领地的进度条件。羁绊分为熟悉（0～39）、默契（40～79）和挚友（80～100）三个岗位档位，编队、背包和设施面板都会显示匹配活动的实际整数奖励。

## 领地

HUD 的“领地”按钮会进入一张可实际游玩的横向基地地图。使用 **A / D**、左右方向键或屏幕左右键移动，靠近设施后按 **E** 或点击“交互”。旧书签或链接访问 `territory.html` 时也会进入这里。

- 新档先在中央遗迹修复免费的主基地，随后永久达到 R1，并开放训练场、守护神庙和工坊分区。
- 领地升阶同时要求远征深度、成功撤离、建设度和资源；已经开放的基地范围不会因其他成长变化而缩回。
- 建筑使用固定且唯一的分区，不需要在界面网格中摆放；升级上限随领地等级提高。
- 训练场、神庙、工坊等设施提供短时基地活动。完成活动可为下一次远征准备攻击、防御、补给或经验加成，出发时一次性消耗；若上阵宠物的基地岗位匹配该设施，活动提示会显示宠物协助并提高产出。
- 工坊和水晶矿会以 50% 基础效率积累资源，库存时长随领地等级提升，需要走到对应设施收取。
- 西侧远征入口直接连接远征场景；从远征返回领地时角色会出现在入口附近，上阵宠物会在基地中跟随。

基地 HUD 会显示当前领地等级、建设度、资源、成长共鸣和下一目标；命运桌会显示同一个长期目标。走近设施后才出现对应操作面板。

“成长共鸣”仍汇总命运、主角、宠物与远征进度，用于解释整体成长方向，但不再直接控制地块开关。实际开放范围以永久领地等级为准。

## 成就

HUD 的“成就”按钮打开长期里程碑页面，入口红色数字表示当前可领取数量。页面可按命运、远征、伙伴和领地筛选；达到目标后会显示“可领取”，可以逐项领取或使用“全部领取”。

里程碑记录历史最高进度：例如曾经上阵完整三宠队、达到某个远征深度或建设度后，即使后来调整编队或拆除建筑也不会失去完成状态。领取后资源立即进入库存，状态变为“已领取”。

## 存档与键盘

| 输入 | 行为 |
| --- | --- |
| WASD / 方向键 | 在远征大地图中移动；领地使用 A / D 或左右方向 |
| E | 靠近远征地点或基地设施后交互 |
| 鼠标 / 触摸战场 | 远征中瞄准并射击 |
| Q | 释放队长宠物的队伍技能 |
| R | 条件允许时使用补给恢复生命 |
| M | 轮换地图追踪目标 |
| B | 在安全行进阶段打开或关闭远征背包 |
| F5 | 保存到槽位 1 |
| F9 | 加载槽位 1 |
| Escape | 远征中打开退出确认；其他场景关闭当前弹窗 |
| Tab | 按浏览器默认顺序移动焦点 |

存档保存在浏览器 LocalStorage。清除站点数据会清空进度；请在设置页面定期导出备份。

## 常见问题

**游戏无法启动**：不要直接双击 HTML。运行 `python3 -m http.server 4174 --directory .` 后访问 `http://localhost:4174`。

**竖屏显示提示**：游戏目前只支持横屏。旋转设备后会自动进入游戏。

**领地资源不一致**：所有领地进度都以主游戏内的领地场景为准。旧 `territory.html` 只负责跳转。
