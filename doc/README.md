# 宠物计划 项目文档

## 项目概述

宠物计划是一个基于 HTML5 Canvas 的宠物养成类游戏，采用纯前端技术栈开发。游戏包含角色系统、战斗系统、升级系统、用户界面等多个功能模块。

## 技术栈

- **前端框架**: 原生 JavaScript (ES6+)
- **渲染引擎**: HTML5 Canvas
- **样式**: CSS3 (包含动画和响应式设计)
- **构建工具**: 无 (纯前端项目)
- **浏览器支持**: 现代浏览器 (Chrome, Firefox, Safari, Edge)

## 核心特性

1. **实时战斗系统** - 基于 Canvas 的 2D 战斗
2. **角色升级系统** - 多属性升级和批量升级
3. **资源管理系统** - 金币、红宝石等游戏货币
4. **响应式UI** - 适配移动端和桌面端
5. **动画效果** - 丰富的视觉反馈和动画

## 快速开始

1. 克隆项目到本地
2. 使用现代浏览器打开 `index.html`
3. 开始游戏体验
