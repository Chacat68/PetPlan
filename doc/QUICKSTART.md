# 快速开始

## 启动游戏

```bash
cd PetPlan
python3 -m http.server 4174 --directory .
```

在浏览器访问 [http://localhost:4174](http://localhost:4174)。项目依赖 ES Modules，不支持直接使用 `file://` 打开。

PetPlan 的游戏舞台固定为横屏 16:9。手机或平板竖屏时会显示旋转设备提示。

## 第一轮玩法

1. 在命运场景点击桌面硬币，获得正面或反面资源。
2. 查看右侧“推荐”商店，优先购买当前可负担的成长项。
3. 购买桌面硬币、小助手或自动结算后，观察命运循环加速。
4. 进入战斗，使用主角和上阵宠物获得金币、红宝石、水晶。
5. 在领地建造建筑，将资源转化为长期属性和产出。

## 快捷键

| 快捷键 | 行为 |
| --- | --- |
| F5 | 保存到槽位 1 |
| F9 | 从槽位 1 加载 |
| Escape | 关闭当前弹窗 |

## 常见问题

**页面提示需要本地服务**：在项目根目录运行 `python3 -m http.server 4174 --directory .`，再打开 `http://localhost:4174`。

**竖屏只显示旋转提示**：这是预期行为。请将设备转为横屏。

**旧的 `territory.html` 链接**：该页面会自动跳转到主游戏内的领地场景，存档和资源与主游戏一致。
