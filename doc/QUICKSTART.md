# 快速开始

## 启动游戏

```bash
cd PetPlan
python3 -m http.server 4174 --directory .
```

在浏览器访问 [http://localhost:4174](http://localhost:4174)。项目依赖 ES Modules，不支持直接使用 `file://` 打开。

PetPlan 的游戏舞台固定为横屏 16:9。手机或平板竖屏时会显示旋转设备提示。

## 第一轮玩法

1. 在命运场景点击桌面硬币，获得正面或反面资源。
2. 查看右侧“推荐”商店，优先购买当前可负担的成长项。
3. 购买桌面硬币、小助手或自动结算后，观察命运循环加速。
4. 进入领地并通过次元探索门开始宠物远征，使用 WASD 或方向键探索大地图和小地图迷雾。
5. 点击顶部目标条或按 M 在 8 个热点间轮换追踪，亲自走到附近后按 E 或点击“交互”；敌人会在环境中持续巡逻，鼠标或触摸战场可随时瞄准射击。
6. 搜索后从候选战利品中手动取舍；背包满时决定替换或放弃，并用锁定与安全袋保护关键物资。
7. 根据生命、弹药、威胁和背包价值决定继续深入，或随时返回西侧入口，守住约 4 秒倒计时完成撤离。
8. 回到整备页后，把战利品投入工坊、容量升级、配装保险、合约或出售，再准备下一局。

## 宠物远征速览

- 每局从西侧入口出发，拥有满生命、2 份补给和默认 8 格背包；整张地图一次生成 8 个热点，探索过的地点仍可回访。
- 小地图只显示已经揭开的迷雾区域。点击顶部目标条或按 M 只轮换追踪方向与距离区间，不会自动寻路；靠近后按 E 或点击“交互”。
- 快速搜索会展示 2–3 件候选，彻底搜刮展示 3–4 件候选且耗时、警戒和伏击风险更高。搜索期间世界继续运行，受击会中断。
- 敌人会在地图中巡逻，通过视线和枪声发现玩家；失去目标后会调查最后位置，搜索一段时间再脱战返巡逻。
- 出发前选择一把武器，进入远征后保持锁定。鼠标或触摸战场可在局内任意时机射击，按 R 手动换弹；枪械完全无弹药时，射击输入会改为短距离弱近战。地图弹药掉落会在靠近时拾取。
- 只有队长宠物提供一个主动队伍技能，其余上阵宠物继续提供攻击、护卫和探索被动。
- 按 4 使用补给恢复生命；安全屋还能回血、降低威胁并补充补给。
- 背包满时不会自动替换：必须在新物品与现有物品之间手动取舍。任务物资可锁定，安全袋容量有限但能在失败或主动退出时保留其中物品；打开背包不会暂停世界。
- 入口信标从开局起即可用，回到入口后约 4 秒完成撤离。离开撤离圈只暂停倒计时，敌人和战斗仍会继续；深入 5 个热点后还可选择高压应急撤离。
- 成功撤离才能带回全部背包和本局战斗收益。Esc 主动退出不等于撤离，战斗金币与经验归零，只回收安全袋物品。
- 基础掉落池有 16 种战利品，其中制作材料按机械、医疗、情报、宠物素材、稀有核心五类用途整理；整备页可制作弹药包、补给、保险券与护甲，升级仓库/背包/安全袋，并按装备售价的 20% 为非永久配装投保。
- 主线按“重建补给线 → 调查核心仓库”两段依次推进；完成后不会重复刷出。

## 快捷键

| 快捷键 | 行为 |
| --- | --- |
| WASD / 方向键 | 在远征大地图中移动 |
| E | 靠近远征地点后交互 |
| 鼠标 / 触摸战场 | 瞄准并射击 |
| Q | 释放队伍技能 |
| R | 手动更换弹匣 |
| 4 | 条件允许时使用补给恢复生命 |
| M | 轮换地图追踪目标 |
| B | 打开或关闭远征背包；世界不会暂停 |
| F5 | 保存到槽位 1 |
| F9 | 从槽位 1 加载 |
| Escape | 远征中打开退出确认；其他场景关闭当前弹窗 |

## 常见问题

**页面提示需要本地服务**：在项目根目录运行 `python3 -m http.server 4174 --directory .`，再打开 `http://localhost:4174`。

**竖屏只显示旋转提示**：这是预期行为。请将设备转为横屏。

**旧的 `territory.html` 链接**：该页面会自动跳转到主游戏内的领地场景，存档和资源与主游戏一致。
