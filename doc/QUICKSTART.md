# 快速开始

## 启动游戏

```bash
cd PetPlan
python3 -m http.server 4174 --directory .
```

在浏览器访问 [http://localhost:4174](http://localhost:4174)。项目依赖 ES Modules，不支持直接使用 `file://` 打开。

PetPlan 的游戏舞台固定为横屏 16:9。手机或平板竖屏时会显示旋转设备提示。

## 第一轮玩法

1. 在命运场景点击桌面硬币，获得正面或反面资源。
2. 查看右侧“推荐”商店，优先购买当前可负担的成长项。
3. 购买桌面硬币、小助手或自动结算后，观察命运循环加速。
4. 进入“战斗”并开始宠物远征，从两条候选路线中选择下一处区域。
5. 在搜索区域选择快速搜索、仔细搜刮或宠物侦察；遇敌后，主角和宠物会自动攻击。
6. 清理至少 3 个区域并回到路线规划后，根据生命、威胁和背包价值决定继续深入或启动撤离。
7. 守住撤离信标，将金币、水晶和经验带回局外，再去领地建造建筑形成长期成长。

## 宠物远征速览

- 每局从满生命、2 份补给和 8 格空背包开始，最多深入 8 个区域。
- 快速搜索风险最低；仔细搜刮收益更高；宠物侦察需要至少上阵 1 只宠物。
- 主角与宠物自动攻击。点击敌人可以锁定目标，画面底部按钮用于释放宠物主动技能。
- 补给可以恢复生命；安全屋还能回血、降低威胁并补充补给。
- 撤离成功才能带回全部背包物品。战败或放弃会遗失背包，只保留部分战斗收益。

## 快捷键

| 快捷键 | 行为 |
| --- | --- |
| F5 | 保存到槽位 1 |
| F9 | 从槽位 1 加载 |
| Escape | 关闭当前弹窗 |

## 常见问题

**页面提示需要本地服务**：在项目根目录运行 `python3 -m http.server 4174 --directory .`，再打开 `http://localhost:4174`。

**竖屏只显示旋转提示**：这是预期行为。请将设备转为横屏。

**旧的 `territory.html` 链接**：该页面会自动跳转到主游戏内的领地场景，存档和资源与主游戏一致。
