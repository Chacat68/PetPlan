# 🚀 快速开始

## 启动游戏

### 方法一：Python（推荐）

```bash
cd PetPlan
python -m http.server 8080
# 访问 http://localhost:8080
```

### 方法二：Node.js

```bash
npx http-server
```

### 方法三：VS Code Live Server

右键 `index.html` → "Open with Live Server"

## 核心玩法

### 1. 角色升级

点击属性旁的 `+` 按钮升级（支持长按快速升级）。

**推荐升级顺序**：攻击力 → 生命值 → 暴击 → 攻速

### 2. 宠物系统

- 点击底部 **宠物** 按钮
- 在 **图鉴** 解锁新宠物
- 在 **编队** 上阵宠物参战

### 3. 领地系统

- 点击底部 **领地** 按钮
- 建造和升级建筑
- 建筑自动产出资源

### 4. 存档管理

- 点击 **💾 存档** 按钮
- 支持 5 个存档槽位
- 支持导出/导入 JSON 备份

## 常见问题

**Q: 游戏无法加载？**

- 使用本地服务器运行（不要直接双击 HTML）
- 使用 Chrome/Firefox 最新版本

**Q: 存档丢失？**

- 定期导出 JSON 备份
- 不要清除浏览器数据
