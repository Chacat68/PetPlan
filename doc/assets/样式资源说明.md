# 样式资源说明

## 概述

Pet Plan 项目使用 CSS3 技术构建现代化的用户界面，包含响应式设计、动画效果、主题色彩等丰富的样式资源。

## 样式文件结构

```
css/
├── style.css                    # 主样式文件
├── menu.css                     # 菜单样式
└── character-management.css     # 角色管理样式
```
