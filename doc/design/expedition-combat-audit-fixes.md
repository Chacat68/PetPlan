# 远征战斗策划自查与整改说明

> 历史阶段记录：当前玩法规则以 [远征玩法精简改版（2026-07-23）](./expedition-gameplay-simplification-20260723.md) 为准。

> 完成日期：2026-07-21
> 适用玩法：宠物远征 · 单局 RPG 搜打撤
> 存档版本：`1.5.0`

## 1. 本次目标与代码基线

本次改动针对远征战斗自查中发现的完整问题清单，一次性完成战斗手感、搜打撤决策、地图探索、宠物成长、长期目标、存档与移动端交互整改。

开始修改前已同步远端当前可见的最新功能提交：

- `7d1a51f feat: 完善宠物闭环与长期里程碑`
- `9856857 feat: 重设计里程碑档案界面`

本轮在上述代码基线上继续开发，没有覆盖远端已有的宠物闭环、里程碑档案馆和领地世界改动。

## 2. 问题与整改结果

| 自查问题 | 整改结果 |
| --- | --- |
| 移动存在双重更新，速度受帧率或模块调用次数影响 | 将远征角色移动统一交给 `CombatSystem.updateHeroMovement()`；游戏核心只保留单一移动入口，离开远征、窗口失焦或弹窗打开时立即清空输入。 |
| 自动攻击配合无限风筝，走位几乎没有输出代价 | 改为手动准星与固定弹道；三类枪械分别管理弹匣、备弹、射速、射程、散布和换弹。移动时散布扩大、伤害和射击节奏下降，允许玩家打空。 |
| 障碍只挡角色、不挡攻击，掩体没有战术价值 | 英雄索敌和射击加入障碍视线检测；投射物命中障碍后消失；怪物碰撞按 X/Y 轴滑动，避免卡死在墙角。 |
| 敌人同质化，首领只是数值更高 | 增加远程敌人、冲锋预警、首领范围技预警与不同攻击节奏；精英数量受威胁和超限压力共同影响。 |
| 撤离可以反复离圈保留进度，守点压力不足 | 离圈仅有 0.75 秒容错，之后撤离进度按 1.4 倍速度回退；敌人占据信标时倒计时暂停；守点期间会持续刷新追兵。 |
| 深入后返程安全，缺少“带货回家”的高压阶段 | 进入返程后按实际移动距离累计返程压力，最多触发 3 档额外威胁；世界事件可提供有限返程减压，但不能完全消除风险。 |
| 威胁只有惩罚，玩家缺少主动承担风险的理由 | 每 25 点威胁提高一档战斗收益，超限压力继续增加奖励、精英与撤离强度；威胁不再在 100 点后失去意义。 |
| 三种搜索方式收益结构接近，宠物侦察容易成为固定最优解 | 三种搜索方式改为清晰角色：快速搜索低暴露、仔细搜刮消耗补给换产量、宠物侦察稳定品质但仍有时间和威胁成本。界面显示精确耗时、物品数、威胁、伏击与补给成本。 |
| 安全屋是无代价回血降威胁，路线选择失去张力 | 休整消耗 1 份补给，恢复 42% 生命并降低威胁；直接离开保留补给，并获得下一战“隐蔽先手”。 |
| 背包满后自动替换，核心搜打撤取舍被系统代做 | 背包溢出时暂停后续行动，玩家手动选择替换某一格或丢弃新物品；每件物品显示用途、价值和稀有度。 |
| 战败与主动放弃收益接近，可能通过放弃刷未完成遭遇奖励 | 战败保留战斗金币 30%、经验 40%；主动放弃仅保留金币 12%、经验 20%。未完成遭遇的临时击杀奖励不会在放弃时提交。 |
| 缺少保险机制，失败时只有全有或全无 | 增加保险格。玩家可在背包中指定保护物品，失败或放弃时只带回保险物；成功撤离不受限制。 |
| 宠物只提供技能或搜索修正，远征成长闭环弱 | 宠物获得远征经验、等级和羁绊；队伍提供移速、攻击、暴击、暴伤、治疗、防伤与护卫值，并可提供一次濒死救援。技能冷却只在实际战斗中推进。 |
| 玩家与领地属性没有完整进入单局，回血/防御契约不一致 | 开局锁定玩家、领地、战备和宠物快照；局内生命接入生命回复、防御、领地护盾、宠物减伤与治疗增幅，避免中途换装或升级改变当前局。 |
| 深层收益与长期目标不足，失败深度还可能解锁领地 | 新增合约残片、深层材料、撤离红宝石、最高携带价值、三宠编队、无伤撤离、首领击杀等长期记录；领地升阶只读取成功撤离深度。 |
| 固定地图和情报提前泄露，探索感不足 | 每局使用种子生成带抖动的地点、障碍和道路；未知地点只显示“未知信号”，靠近后才揭示身份；加入一次性世界事件与探索里程碑。 |
| 刷新页面会清空长局，移动端按钮偏小 | 存档版本升级至 `1.5.0`，保存路线、世界、坐标、生命、冷却、怪物、枪械、固定弹丸、搜索读条、撤离类型、仓库和配装等状态；页面隐藏时自动保存。方向键和关键操作按钮最小触控尺寸为 44px。 |

## 3. 当前核心规则

### 3.1 搜索决策

| 方式 | 耗时 | 战利品 | 威胁 | 伏击率 | 补给成本 | 定位 |
| --- | ---: | ---: | ---: | ---: | ---: | --- |
| 快速搜索 | 3 秒 | 1 件 | +3 | 4% | 0 | 低暴露、快速离场 |
| 仔细搜刮 | 12 秒 | 2～3 件 | +14 | 30% | 1 | 用补给换最大产量 |
| 宠物侦察 | 7 秒 | 2 件 | +8 | 10% | 0 | 稳定品质与较低伏击 |

宠物探索天赋仍可修正结果，但品质、额外物品、威胁、补给率和伏击率均受统一上限约束，避免单一队伍组合垄断最优答案。

### 3.2 战斗与补给

- 主角按准星方向手动射击，宠物仍自主协同；移动射击存在散布、射速和伤害惩罚。
- 障碍阻挡角色、怪物与双方投射物；敌方远程弹丸可被侧移躲避，准星与短瞄准线提供固定弹道反馈。
- 补给恢复最大生命的 32%，使用后有 5 秒冷却；受到伤害后的 1.1 秒安全窗口内不能立即使用。
- 局内生命会持续应用玩家生命回复；宠物护卫优先吸收部分伤害，随后应用队伍减伤。
- 冲锋与首领范围技有预警，玩家需要通过走位处理，而不是只比较数值。

### 3.3 威胁、返程与撤离

- 威胁每 25 点提升一个警戒档位，并同步提高战斗收益。
- 100 点以上转为“超限压力”，继续提高精英数量、撤离时长、追兵频率与奖励倍率。
- 解锁撤离后，返程移动距离会累积额外压力；越深入、带货越多，回程阶段越危险。
- 撤离圈允许 0.75 秒短暂误差；持续离圈会倒退进度，敌人争夺信标时暂停推进。
- 撤离战会持续补充追兵，不能靠清掉首波后挂机完成。

### 3.4 背包、保险与结算

- 背包默认 8 格。满包发现新物品时，必须手动选择替换或丢弃。
- 保险格可保护指定物品；失败或放弃只保留已投保物品。
- 成功撤离带回全部战斗收益、背包物品、合约残片、深层材料和符合条件的红宝石。
- 战败保留临时战斗金币的 30% 与经验的 40%；主动放弃降低为 12% 与 20%。
- 未完成遭遇的击杀和临时奖励不会因主动放弃而结算。
- 所有结算仍由 `settled` / `lastSettlement` 做幂等保护，防止重复发奖。

## 4. 存档与兼容

存档版本由 `1.4.0` 升级至 `1.5.0`。除长期记录外，活动远征现在还会保存：

- 路线、深度、搜索状态、威胁、超限压力、补给、背包、保险与待处理的背包选择；
- 世界种子、地点、障碍、迷雾、事件和已发现状态；
- 主角坐标、局内生命、属性快照、返程压力和战备；
- 怪物、当前遭遇、待生成队列、临时奖励和投保结果；
- 宠物队伍快照、护卫值、救援状态、技能冷却；
- 撤离剩余时间、离圈计时和增援计时。
- 枪械模式、弹匣、备弹、换弹、准星和飞行中的固定弹丸；
- 容器搜索读条、双撤离点状态与当前撤离类型；
- 局外仓库、待领取箱、配装、合约、活动配装快照和幂等结算账本。

载入顺序调整为先恢复宠物，再恢复战斗，确保活动远征中的队伍实例和技能冷却能找到正确宠物。旧存档仍兼容 `bestWave`、`victories`、`defeats` 等旧字段；缺失的新字段使用保守默认值。

## 5. 主要改动文件

- `js/modules/combat-system.js`：战斗、走位、敌人行为、撤离、结算、属性快照和局内存档。
- `js/modules/expedition-run-system.js`：搜索、威胁、背包、保险、安全屋、事件效果与结算规则。
- `js/modules/expedition-world-system.js`：种子地图、迷雾、未知地点、世界事件和世界存档。
- `js/modules/expedition-meta-system.js`：仓库、配装风险、合约、奖励余额、旧计数迁移和结算账本。
- `js/modules/pet-system.js`：远征经验、等级、羁绊、角色光环、护卫和救援。
- `js/modules/achievement-config.js`、`achievement-system.js`：远征长期里程碑。
- `js/modules/territory-system.js`、`progression-config.js`：成功撤离深度门槛和跨系统目标。
- `js/modules/save-system.js`：`1.5.0` 存档、迁移与“先 Meta、后活动战斗”的恢复顺序。
- `js/controllers/battle-scene-controller.js`、`index.html`、`css/style.css`：风险预览、搜索预览、保险、背包抉择、宠物护卫和移动端交互。
