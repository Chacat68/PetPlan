# 远征连续搜打撤重构说明（2026-07-24）

## 定位

本轮将宠物远征从“选择节点后进入独立战斗”的流程，重构为一张持续运行的大地图搜打撤。设计参考《逃离鸭科夫》的核心张力，但保留 PetPlan 的轻量操作、宠物队伍和短局节奏：玩家只需要移动、瞄准、换弹、补给、交互和取舍，不引入复杂身体部位、弹匣装填或多层容器管理。

目标体验是：

1. 地图从开局就是完整空间，路线选择不删除其他机会。
2. 战斗由敌人的实际视听感知触发，而不是由界面阶段触发。
3. 搜索、整理背包和撤离都有真实时间风险。
4. 任何收获都必须经过“继续深入还是现在回家”的决策。
5. 战利品有清晰用途，并能反哺下一局，而不是只换成结算分数。

## 单局结构

### 完整地图与 8 个热点

每局在 `3000×1900` 世界中一次生成以下 8 个热点：

| 顺序 | 区域 | 类型 | 主要作用 |
| --- | --- | --- | --- |
| 1 | 西侧补给带 | 搜索 | 低压获取基础物资 |
| 2 | 北部巡逻线 | 战斗 | 建立首次交战压力 |
| 3 | 南部安全屋 | 营地 | 回血、降警戒、补给 |
| 4 | 旧城仓储区 | 核心仓库 | 中风险搜索，也是第二段主线地点 `core-vault` |
| 5 | 中部封锁线 | 战斗 | 中段火力与弹药检查 |
| 6 | 东部高危区 | 精英 | 高风险稀有战利品 |
| 7 | 深区物资带 | 搜索 | 深区高品质补充 |
| 8 | 核心守卫区 | 首领 | 单局最高风险与收益 |

所有热点在开局同时存在。追踪只改变导航目标，不会关闭其他热点；完成地点会记录完成状态和访问次数，允许回访，但不会重复发放事件奖励。地图仍通过迷雾逐步揭示，导航只提供方向、区域和距离区间，不显示精确米数或自动寻路。

### 持续敌人与脱战

环境敌人从开局起分布在大地图中，并持续更新以下状态：

```text
巡逻
  -> 看见玩家或听见枪声
调查
  -> 确认玩家位置
交战
  -> 失去视野
搜索最后已知位置
  -> 超时且没有新声源
返回巡逻
```

感知遵循两个来源：视线用于确认当前目标，枪声用于暴露一个调查位置。敌人失去视野后不会继续获得玩家的精确坐标，而是检查最后已知位置；玩家拉开距离、利用障碍并停止制造噪声后可以脱战。`raidThreatActive` 只表示当前确有敌人处于警觉状态，不等同于“远征正在进行”。

地点遭遇仍可生成额外敌人，但战斗结束后只恢复自由探索，不清空地图上的环境敌人，也不强制完成未满足条件的热点。

## 操作与战斗

| 输入 | 行为 |
| --- | --- |
| WASD / 方向键 | 八方向移动，斜向速度归一化 |
| 鼠标 / 触摸战场 | 任意局内阶段瞄准并射击 |
| E | 近距地点交互、搜索或启动撤离 |
| Q | 释放队长宠物的主动技能 |
| R | 手动换弹 |
| 4 | 消耗一份补给恢复生命 |
| M / 顶部目标条 | 轮换追踪热点、事件或撤离点 |
| B | 打开/关闭背包；世界不会暂停 |
| Esc | 打开退出确认 |

武器仍采用步枪、霰弹枪、精确枪三种轻量差异。射击不受 `combat` 阶段限制；玩家可以在巡逻敌人尚未发现自己时先手攻击，也可以在撤离守点时持续还击。

弹药分为弹匣与备弹，换弹改为玩家按 `R` 主动执行。环境或敌人可产生弹药拾取物，玩家靠近后自动补充当前武器备弹。弹匣和备弹都耗尽时，射击输入切换为有冷却的短距离弱近战，保证不会因零弹药完全锁死，但效率明显低于正常武器。

## 搜索与背包风险

### 搜索

| 方式 | 时长 | 候选数 | 品质 | 风险 |
| --- | ---: | ---: | --- | --- |
| 快速搜索 | 约 2 秒 | 2–3 件 | 基础 | 警戒 +2，低伏击率 |
| 彻底搜刮 | 约 5 秒 | 3–4 件 | 提升 | 警戒 +10，伏击率显著更高 |

搜索计时期间世界继续运行，敌人可以巡逻、调查和攻击；玩家受击会中断搜索，本次不产生战利品。宠物探索天赋作为被动修正候选、补给率或伏击率，不增加额外操作按钮。

浅层掉落受稀有度保护：前两区史诗品质接近零，稀有品质约为 5%，避免开局短途搜索取代深入地图的价值。

### 手动取舍、锁定与安全袋

背包默认 8 格，容量可通过局外设施升级。搜索生成的是候选战利品，而不是自动入包结果：

- 有空格时按顺序收入背包。
- 背包已满时进入待选队列，玩家选择替换某件物品，或放弃新物品并保留当前背包。
- 系统不再按单一分数自动替换低价值物品。
- 玩家可锁定重要物品；任务物默认带任务标记，锁定物、任务物和安全袋物品不能成为普通替换目标。
- 打开背包不会暂停世界；正在搜索、处理待选项或撤离倒计时时不能另开背包抽屉。

安全袋拥有独立容量。放入其中的局内战利品在战败或主动退出时仍能回收；它不保护普通背包，也不等同于出发装备保险。

## 撤离与失败

### 入口撤离

西侧入口信标从开局即开放，不再要求先清理 3 个地点。玩家可在任意收益阶段原路返回，近距交互后进行约 4 秒倒计时。入口撤离不消耗补给，守点压力较低，但仍可能被环境敌人或增援打断节奏。

### 应急撤离

深区应急点在探索深度达到 5 后开放。它免去长距离折返，但消耗 1 份补给，倒计时更长且敌人更密集。入口和应急点分别服务“稳妥返程”与“高压省路”两种决策。

两类撤离都只在玩家留在信标圈内时推进倒计时。离圈不会重置或倒退进度，只会暂停；敌人、投射物和世界逻辑照常运行。

### 结算边界

- 成功撤离：带回普通背包与安全袋物品，并结算本局战斗金币、经验、水晶和其他奖励。
- 主动退出：不结算战斗金币或经验，普通背包遗失，只回收安全袋物品。
- 战败：按战败保底结算，未保护战利品和未保险的非永久配装遗失；安全袋物品与已保险配装回收。

主动退出的目的只是处理无法继续操作的局面，不提供“打到一半直接领奖”的止损捷径。所有结算保持幂等，同一局不得重复发奖。

## 战利品与局外用途

### 16 种基础战利品

| 品质 | 战利品 |
| --- | --- |
| 普通 | 破旧钱袋、能量零件、旧城区情报、通用弹药盒、简易绷带、旧铜线束 |
| 优秀 | 精制宠粮、战地医疗包、完整机械芯、轻型护甲片、封锁区通行证 |
| 稀有 | 秘境水晶、异兽铭牌、复合防护背心、密封研究样本 |
| 史诗 | 星辉核心 |

基础材料按五种核心用途归档：

| 用途 | 说明 |
| --- | --- |
| 机械件 | 武器维护、弹药与基础护甲制作 |
| 医疗物资 | 行动补给与护甲内衬制作 |
| 情报 | 主线调查、保险与路线准备 |
| 宠物素材 | 伙伴补给与宠物装置制作 |
| 稀有核心 | 高阶装备、保险券与基地扩容 |

装备可带入远征但承担失败遗失风险；行动物资在出发时转化为补给或备弹；保险凭证用于承保一件非永久配装；贵重品主要用于出售换取金币。物品卡直接显示用途，避免所有物资最终都只看出售价格。

### 工坊配方

| 配方 | 消耗 | 产出 |
| --- | --- | --- |
| 通用弹药包 | 机械件 ×2 | 下一局额外备弹 |
| 伙伴行动补给 | 医疗物资 ×1、宠物素材 ×1 | 下一局补给 +1 |
| 装备保险券 | 情报 ×2、稀有核心 ×1 | 一次免费装备承保 |
| 拼装野战护甲 | 机械件 ×3、医疗物资 ×1 | 6 点防御的可遗失护甲 |

工坊制作只在没有活动远征时开放。仓库满时，制作产物进入待领取箱，避免静默丢失。领地工坊原有的被动金币生产降为 12% 基础效率；它与远征材料制作是两套并行功能。

### 容量升级

- 仓库扩建：3 级，容量依次增加 8、8、12。
- 行动背包：2 级，每级增加 1 格。
- 安全袋：2 级，每级增加 1 格。

升级消耗五类材料，且只在局外进行。新背包与安全袋容量在下一次 `startRun()` 时注入单局，活动远征不会中途扩容。

### 装备保险

非永久配装可在出发前投保一局。保费为该装备出售价格的 20%，向上取整且至少 1 金币；也可消耗 1 张装备保险券。取消保险不会返还已支付保费。开局时保险列表被锁定为本局快照，失败结算后返还受保装备。

装备保险与安全袋的边界如下：

- 装备保险保护“出发时带入的非永久装备”。
- 安全袋保护“本局搜索到并主动放入的战利品”。

### 两段主线

主线合约固定按顺序推进且只完成一次：

1. **重建补给线**：带回并交付 3 件机械类物资。
2. **失联仓库**：调查类型为 `cache`、地点 ID 为 `core-vault` 的旧城核心仓库。

第二段同时校验地点 ID 与类型，避免其他缓存点误触发。主线完成后不重复生成。支线委托只在一次远征结算后刷新周期；接受、放弃或交付不会立即无限刷新委托板。

## 存档与兼容

活动远征快照需要保留：

- 8 个热点及其完成、访问状态；
- 玩家位置、迷雾、生命、警戒、补给和武器弹药；
- 背包、物品锁定、安全袋与待选队列；
- 环境敌人位置、生命、感知状态和最后已知目标；
- 世界弹药拾取、搜索进度与撤离倒计时。

局外存档需要保留：仓库、配装、材料用途、容量等级、已选保险、合约板、主线历史和结算幂等记录。旧存档缺少新字段时使用默认 8 格背包、1 格安全袋和 0 级容量升级，并把历史可保护物映射到安全袋语义。

## 验收标准

1. 新局始终生成 8 个热点；完成任一点后其他热点仍存在，回访不会重复奖励。
2. 环境敌人可巡逻、因视线或枪声调查、失去目标后搜索并返回巡逻；不会持续获得玩家全图精确坐标。
3. 探索阶段也能射击；`R` 手动换弹、`4` 使用补给，完全无弹时可用弱近战，靠近弹药掉落可补充备弹。
4. 快速搜索只生成 2–3 件候选，彻底搜刮只生成 3–4 件；搜索受击会中断。
5. 满包不会自动替换；锁定物、任务物与安全袋物品不能成为普通替换目标。
6. 入口撤离开局可用且基础倒计时约 4 秒；离圈只暂停，世界继续运行。
7. 主动退出的战斗金币和经验均为 0，只回收安全袋物品。
8. 16 种基础战利品均显示用途；五类材料可实际用于工坊或容量升级。
9. 保费等于装备售价的 20% 或消耗保险券；失败能返还已保险装备。
10. 主线严格按两段顺序推进，核心仓库调查不会被同类但不同 ID 的地点误触发。
