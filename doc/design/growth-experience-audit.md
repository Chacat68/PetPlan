# 成长体验复评

## 结论

PetPlan 的系统本身已经具备成立的成长闭环，但原首局体验没有让玩家完整经历这个闭环。命运桌的输入、自动化、远征的风险收益、领地的永久沉淀和宠物的跨场景价值分别成立；主要问题出在首次呈现顺序、跨场景交接和“临时收益/永久收益”的解释。

本轮把首局主线统一为：

```text
主动翻转 → 扩充容量 → 建立自动化 → 远征探索 → 成功撤离 → 领地沉淀 → 回到下一轮成长
```

这条路径与长期产品循环保持一致，不再从助手直接跳过远征去修基地。

## 逐系统评估

| 系统 | 成长体验判断 | 主要依据 | 本轮处理 |
| --- | --- | --- | --- |
| 命运翻转 | 成立 | 单次操作立即得到正/反面反馈，输入与资源来源清楚 | 首步高亮真实硬币，并解释两类资源用途 |
| 扩充桌面 | 成立 | 增加可操作目标，既有数值成长也有可见变化 | 明确为“提高每轮手动效率”，避免只说硬币 +1 |
| 助手自动化 | 成立 | 从主动点击过渡到持续收益，是早期明显的能力跃迁 | 单独作为第三阶段，先于芯片和速度 |
| 商店推荐 | 基本成立 | 有可负担状态、精确缺口和前后对比 | 引导只定位当前关键卡片，不复制购买逻辑 |
| 远征探索 | 系统成立、首局缺位 | 搜、打、撤有真实风险决策，但原四步目标完全跳过 | 新增首次探索与首次撤离两阶段，并按远征状态实时讲解 |
| 撤离结算 | 成立但需要强调 | 临时背包与永久入库是玩法张力核心 | 在返程和守点阶段持续解释“未撤离不入库” |
| 领地建设 | 成立 | 空间开放、建筑效果和永久等级不会回退 | 放到首次撤离之后，作为收益沉淀的结果而非孤立功能 |
| 宠物 | 成立 | 同时影响战斗、搜索天赋、基地岗位与羁绊 | 在搜索、战斗、领地说明中按使用场景揭示，不在首屏一次灌输 |
| 成就 | 成立 | 只记录长期里程碑，不与教程争夺任务语义 | 保持奖励独立，引导完成不额外造币 |
| 成长倾向 | 成立 | 只影响推荐，不形成不可逆选择 | 首轮结束时说明倾向是建议而不是职业锁定 |

## 原体验的关键断点

1. 原目标顺序为“翻转 → 扩桌 → 助手 → 领地”，与产品定义的“战斗收益 → 领地建设”相冲突。
2. “下一目标”能说明做什么，却不会把视线落到真实交互对象，新玩家仍需自己在高信息密度界面中搜索。
3. 远征内部一次出现路线、移动、搜索、技能、背包、威胁和撤离，缺少按状态逐层揭示。
4. 场景入口从开局全部可见，保留自由是合理的，但需要软引导帮助玩家理解当前优先级。
5. 首局进度此前只在折叠后的目标详情中显示，可发现性不足，也无法在远征和领地中继续承接。

## 新手引导原则

- 不锁场景、不屏蔽真实按钮、不代替玩家做决定。
- 每次只解释一个当前决策，同时回答“做什么”和“为什么”。
- 进度从系统状态派生，乱序完成、读取存档或场景切换都不会卡死。
- 远征说明跟随实际阶段，不用静态教程页提前灌输全部规则。
- 引导可跳过，并可从 HUD 的“指引”入口随时重播。
- 引导状态进入统一存档；旧存档不会被强制弹出新版教程。
- 完成引导不额外发奖励，避免与成就和玩法收益形成重复结算。

## 后续验证指标

如果进入真实数据阶段，应优先观察：

- 新玩家从首次翻转到第二枚硬币、首个助手的耗时分布。
- 进入远征后成功完成第 1 个区域的比例。
- 解锁撤离后能否找到西侧信标，以及首次撤离成功率。
- 首次撤离后进入领地并修复主基地的比例。
- 引导跳过率、重播率和各阶段停留时长。

在没有行为数据前，不建议继续同时调整助手成本、掉落概率和领地门槛；先验证新的完整路径，再决定数值节奏是否需要收紧。
